我在這個 task 中理解到的 git 是什麼東西
=======================================

git 就是一個帶有**時空錯亂**功能的版本控制系統。
--------------------------------------------

  以下以列表方式簡述學習印象
  - 能`分支`出數個宇宙後各自發展，再 merge 成理想中的樣子
  - 能任意跳轉到不同版本
  - 兩個版本合併時如果同一區塊都被修改過，則會發生衝突（conflict）需要手動擇一或者另做修改
  - 結合 GitHub 方便許多人**一起開發**同一個專案，而且遠端
  - 善用`註解`和`標籤`將會*上天堂*

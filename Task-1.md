我在這個 task 中理解到的 git 是什麼東西
=====
  * *git* 為一種可以管理檔案的控制系統。
  * 操作者可以在本機端利用 git 指令控管檔案。
  * 檔案修改的歷史紀錄只要有 commit 就會被 git 儲存在本機端的 repository。
  * 每個 commit 就像是在檔案上貼標籤，紀錄著檔案 commit 時的狀態。
  * 倘若變更檔案後無 commit，git 的 repository 就不會紀錄此次檔案變更。

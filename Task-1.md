#  我在這個 task 中理解到的 git 是什麼東西
  - git是一個專案、一個項目
    - 指令以git為主詞，後面接動詞，表達要進行的動作
    - 像是：
      - git init表示由零開始
      - git add 檔案名稱 表示要將檔案匯入git
      - git status則是查看、確認git當前的狀態
